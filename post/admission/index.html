中学受験

Fri, Oct 1, 2004 in 時代背景 using tags

2004年（当時小6）10月、中学受験準備のためにWebサイトの管理の継続を断念しました。

しかし、親に隠れてPCでサイト管理をしたり遊んでいたりしたため学業に身が入らず、受験勉強に大きな影響が出ました。

受験の結果、中高一貫校に進むことになりました。
